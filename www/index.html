Paper Playground

Interactive Interaction Design

To run the tool, open the three pages below:

  1. Camera - Configure your paper detection, change spaces (sets of programs), and access the program editor here!
  2. Projector - Required for code execution. Optionally, this window can be used to verify program recognition or display web canvas graphics or web audio.
  3. Design Board - Interactive Design Board. The multimodal components you create using PhET Libraries in your programs will appear here!
  4. Creator - A tool to help create programs and write code.
  5. CreatorAI - An AI powered tool to create paper programs.

Instructions and tutorials for use are currently under construction. The best documentation at this time is the original paperprograms tutorial.md. Documentation updates coming soon!

This tool was adapted from Paper Programs by JP Posma, which was inspired by the work at Dynamicland.

Find the Paper Playground source code and documentation in the paper-land GitHub repository.
